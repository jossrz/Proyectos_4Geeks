# Forensic Analysis & Infrastructure Hardening Project Auditoría de seguridad integral que abarca desde la investigación forense hasta el endurecimiento del sistema.
Forensics: Reconstrucción de líneas de tiempo de ataques y análisis de incidentes mediante Autopsy y logs.
Offensive: Evaluación de vulnerabilidades y escalada de privilegios con Nmap y Metasploit.
Defensive: Implementación de Firewalls, Fail2ban y monitoreo activo con Wazuh.
Compliance: Alineación con los marcos ISO 27001 y NIST.
